Michael Pilosov
Dec 2, 2018

More description to come soon.

I am reviving an old project that I have presented at a few conferences as an undergraduate. I won a cash prize for a talk I gave demonstrating an early version of this which involved my first graphical user interface.

I am now transcribing work into Python to get away from the limitations of Matlab’s licensing structure.
It is a lengthy process. Please be patient.
